# Wisconsin Council of Churches — Judicatory Data Analysis

> **End-to-end data analytics project for a nonprofit client: data cleaning, Excel workbook design, and an interactive HTML dashboard delivered to executive leadership.**

---

## Project Summary

Contracted as a Data Analyst Intern with the Wisconsin Council of Churches (WCC) to solve a real organizational problem: leadership lacked visibility into how member judicatories — regional bodies of Christian denominations — overlapped geographically across Wisconsin's 72 counties.

I owned the full project lifecycle: scoping the data problem, cleaning and structuring raw inputs, building the analytical model, and delivering a client-ready interactive dashboard.

---

## Technical Deliverables

### Multi-Sheet Excel Workbook
- Ingested and normalized inconsistent source data across multiple judicatory records
- Designed a relational multi-sheet structure linking judicatory membership, county coverage, and overlap flags
- Built lookup formulas and summary views to surface actionable patterns without manual cross-referencing

### Interactive HTML Dashboard
- Built a browser-based dashboard in HTML, CSS, and JavaScript — zero dependencies, no install required
- Implemented dynamic filtering so the Executive Director could explore data by county, denomination, or region
- Prioritized usability for a non-technical stakeholder: clean layout, clear labels, accessible design
